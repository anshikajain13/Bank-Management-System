# 🏦 Bank Management System

A modern Bank Management System built using Python and Streamlit.

## Features

- Create Account
- Deposit Money
- Withdraw Money
- View Account Details
- Update Account Information
- Delete Account
- Dashboard with Account Statistics

## Technologies Used

- Python
- Streamlit
- JSON Database

## Project Structure

```text
Bank-Management-System/
│
├── app.py
├── bank.py
├── main.py
├── README.md
├── .gitignore
└── database.json
```

## Run Locally

Clone the repository:

```bash
git clone https://github.com/anshikajain13/Bank-Management-System.git
```

Install dependencies:

```bash
pip install streamlit
```

Run the application:

```bash
streamlit run app.py
```
